# Browser Game 1

This is my first project for learning browser-based game development with Three.js.

The goal of this project is to get familiar with the core concepts and syntax used in browser game development, especially how Three.js works with TypeScript and Vite. It is mainly a practice project for experimenting, learning, and building comfort with the basics.

## What I Am Learning

- Setting up a browser game project with Vite and TypeScript
- Creating and rendering a Three.js scene
- Organizing game code into core, entity, and system modules
- Handling player input
- Working with simple physics, collisions, obstacles, and spawning
- Getting familiar with the syntax and structure of Three.js

## Tech Stack

- Three.js
- TypeScript
- Vite

## Getting Started

Install dependencies:

```sh
npm install
```

Start the development server:

```sh
npm run dev
```

Build the project:

```sh
npm run build
```

Preview the production build:

```sh
npm run preview
```

## Project Notes

This project is not meant to be a finished game yet. It is a learning space for understanding how browser games are structured, how Three.js scenes are built, and how common game development ideas fit together in code.
